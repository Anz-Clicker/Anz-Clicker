Anz Clicker Portable 1.2.0
==========================

How to run:
1. Open this folder.
2. Double-click "Anz Clicker.exe".

No Python installation is required. The portable folder includes the Python runtime,
Qt GUI runtime, app icons, starter scripts, and OCR/Tesseract files used by screen
text actions.

Portable data:
- Saved scripts are stored in the "scripts" folder by default.
- App settings and action defaults are saved next to the executable.
- Keep the files and folders in this directory together when sharing the app.

If Windows SmartScreen appears, choose "More info" and then "Run anyway" if you
trust the source of this build.
