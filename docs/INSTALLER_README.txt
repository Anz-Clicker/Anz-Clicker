ANZ CLICKER - WINDOWS INSTALLATION
==================================

Run "Anz Clicker Setup v<version>.exe" and follow the installer prompts.
Python is not required; the application includes its runtime dependencies.

Program files are installed under Program Files. Files created or customized
by the user are stored separately:

  Scripts:
    %LOCALAPPDATA%\Anz Clicker\scripts

  Settings, custom actions, captures, and future licenses:
    %LOCALAPPDATA%\Anz Clicker\user-data

Installing a newer version or uninstalling Anz Clicker does not remove those
folders. This protects scripts and customization across application updates.
